
## Resumo rápido
- Função é um bloco de código identificado por um nome que pode receber parâmetros.
- Torna o código mais legível e permite o reaproveitamento de lógica.
- Programar com funções é programar de forma **estruturada**.

---

## Definindo uma função

```python
def nome_da_funcao(parametro):
    # bloco de código
    return valor
```

- `def` → palavra reservada para definir uma função.
- `return` → retorna um valor ao final da execução.
- Toda função retorna `None` por padrão se não houver `return`.

### Exemplo básico
```python
def exibir_mensagem():
    print("Olá mundo!")

def exibir_mensagem_2(nome):
    print(f"Seja bem vindo {nome}!")

def exibir_mensagem_3(nome="Anônimo"):
    print(f"Seja bem vindo {nome}!")

exibir_mensagem()
exibir_mensagem_2(nome="Guilherme")
exibir_mensagem_3()
exibir_mensagem_3(nome="Chappie")
```

---

## Retornando valores

- Usa-se `return` para retornar um valor.
- Em Python, uma função pode retornar **mais de um valor** ao mesmo tempo.

```python
def calcular_total(numeros):
    return sum(numeros)

def retorna_antecessor_e_sucessor(numero):
    antecessor = numero - 1
    sucessor = numero + 1
    return antecessor, sucessor

calcular_total([10, 20, 34])          # 64
retorna_antecessor_e_sucessor(10)     # (9, 11)
```

## Argumentos nomeados

- Funções podem ser chamadas com argumentos no formato `chave=valor`.

```python
def salvar_carro(marca, modelo, ano, placa):
    print(f"Carro inserido! {marca}/{modelo}/{ano}/{placa}")

salvar_carro("Fiat", "Palio", 1999, "ABC-1234")
salvar_carro(marca="Fiat", modelo="Palio", ano=1999, placa="ABC-1234")
salvar_carro(**{"marca": "Fiat", "modelo": "Palio", "ano": 1999, "placa": "ABC-1234"})
```

## `*args` e `**kwargs`

- `*args` → recebe múltiplos argumentos posicionais como **tupla**.
- `**kwargs` → recebe múltiplos argumentos nomeados como **dicionário**.

```python
def exibir_poema(data_extenso, *args, **kwargs):
    texto = "\n".join(args)
    meta_dados = "\n".join([f"{chave.title()}: {valor}" for chave, valor in kwargs.items()])
    mensagem = f"{data_extenso}\n\n{texto}\n\n{meta_dados}"
    print(mensagem)

exibir_poema("Zen of Python", "Beautiful is better than ugly.", autor="Tim Peters", ano=1999)
```

## Parâmetros especiais

Controlam como os argumentos podem ser passados para a função.

### Positional only `/`
- Argumentos antes de `/` só podem ser passados por posição.

```python
def criar_carro(modelo, ano, placa, /, marca, motor, combustivel):
    print(modelo, ano, placa, marca, motor, combustivel)

criar_carro("Palio", 1999, "ABC-1234", marca="Fiat", motor="1.0", combustivel="Gasolina")  # válido
criar_carro(modelo="Palio", ...)  # inválido
```

### Keyword only `*`
- Argumentos após `*` só podem ser passados por nome.

```python
def criar_carro(*, modelo, ano, placa, marca, motor, combustivel):
    print(modelo, ano, placa, marca, motor, combustivel)

criar_carro(modelo="Palio", ano=1999, placa="ABC-1234", marca="Fiat", motor="1.0", combustivel="Gasolina")  # válido
criar_carro("Palio", ...)  # inválido
```

### Positional and keyword only `/` + `*`
- Combina os dois modos.

```python
def criar_carro(modelo, ano, placa, /, *, marca, motor, combustivel):
    print(modelo, ano, placa, marca, motor, combustivel)

criar_carro("Palio", 1999, "ABC-1234", marca="Fiat", motor="1.0", combustivel="Gasolina")  # válido
```

## Funções como objetos de primeira classe

- Em Python, funções são objetos e podem ser:
  - Atribuídas a variáveis.
  - Passadas como parâmetro para outras funções.
  - Usadas em estruturas de dados.
  - Retornadas por outras funções (**closures**).

```python
def somar(a, b):
    return a + b

def exibir_resultado(a, b, funcao):
    resultado = funcao(a, b)
    print(f"O resultado da operação {a} + {b} = {resultado}")

exibir_resultado(10, 10, somar)
# O resultado da operação 10 + 10 = 20
```

---

## Escopo local e global

- Dentro de uma função, o escopo é **local**.
- Alterações em objetos imutáveis são perdidas ao sair da função.
- Para usar uma variável global dentro de uma função, use `global`.

```python
salario = 2000

def salario_bonus(bonus):
    global salario
    salario += bonus
    return salario

salario_bonus(500)  # 2500
```

### Observação importante
- O uso de `global` **não é uma boa prática**.
- Prefira retornar valores e reatribuir fora da função.

---

## Erros comuns
- Esquecer o `return` quando a função precisa devolver um valor.
- Confundir escopo local com global.
- Usar `global` desnecessariamente.

## Boas práticas
- Funções devem ter **uma única responsabilidade**.
- Use nomes descritivos no padrão `snake_case`.
- Prefira `return` a modificar variáveis globais.
- Use parâmetros com valores padrão quando fizer sentido.

# Python - Funções (complemento)

## Resumo rápido
- Funções são blocos de código reutilizáveis.
- Uma função pode ser **chamada**, **retornar valor** e até usar **variáveis globais**.
- Python também possui várias **funções built-in** úteis.

---

## Chamada de função

```python
function_name()
```

### Exemplo
```python
print()
```

- A chamada de função executa o bloco associado ao nome da função.

---

## Criação de função

```python
def function_name(parâmetro):
    function_body
```

### Exemplo
```python
def chato_function():
    print("Olá!")
```

---

## Palavra-chave `return`

- `return` faz a função **devolver um valor**.
- Se não houver valor explícito, a função retorna `None`.

```python
def minha_funcao():
    return 13

x = minha_funcao()
print(x)  # 13
```

```python
def sem_retorno():
    return

print(sem_retorno())  # None
```

### Observação
- Quando uma função retorna um valor, esse valor pode ser armazenado em uma variável.

---

## Variável global com `global`

- `global` indica que a variável usada dentro da função é a mesma variável definida fora dela.

```python
saldo = 100

def alterar_saldo():
    global saldo
    saldo += 50

alterar_saldo()
print(saldo)  # 150
```

### Observação
- O uso de `global` deve ser evitado quando possível.
- Em geral, é melhor retornar valores e atribuir fora da função.

---

## Função `print()`

- `print()` exibe informações na tela.

### Parâmetro `end`
- Define o que será colocado no final da impressão.
- Por padrão, termina com quebra de linha.

```python
print("Olá", end="...")
print("mundo")
# Saída: Olá...mundo
```

### Parâmetro `sep`
- Define o separador entre os valores impressos.
- Por padrão, o separador é um espaço.

```python
print("A", "B", "C", sep="-")
# Saída: A-B-C
```

### Print em várias linhas

```python
print("""
Linha 1
Linha 2
Linha 3
""")
```

---

## Funções built-in úteis

### `max()`
- Retorna o maior valor de uma sequência.

```python
maior = max(10, 20, 5, 99)
print(maior)  # 99
```

### `min()`
- Retorna o menor valor de uma sequência.

```python
menor = min(10, 20, 5, 99)
print(menor)  # 5
```

### `round()`
- Arredonda um número.

```python
valor = round(10.6)
print(valor)  # 11
```

### `range()`
- Gera uma sequência de números inteiros.

```python
numeros = range(0, 10, 2)
print(list(numeros))  # [0, 2, 4, 6, 8]
```

### `type()`
- Mostra o tipo de uma variável.

```python
print(type(10))      # <class 'int'>
print(type("abc"))   # <class 'str'>
```

### `len()`
- Mostra o tamanho de uma sequência.

```python
print(len("Python"))   # 6
print(len([1, 2, 3]))  # 3
```

---

## Exemplo de invocação e retorno

```python
def chato_function():
    return 13

x = chato_function()
print(x)
```

- A função é **invocada** com `()`.
- O retorno pode ser salvo em uma variável.
- Nesse caso, `x` recebe o valor `13`.

---

## Resumo final
- `def` cria uma função.
- A função é chamada com `nome_da_funcao()`.
- `return` devolve um valor.
- `global` permite acessar/modificar variável global dentro da função.
- `print()` aceita `end` e `sep`.
- Funções úteis: `max`, `min`, `round`, `range`, `type` e `len`.

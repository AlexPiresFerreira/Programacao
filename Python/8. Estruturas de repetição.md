## Resumo rápido
- O `for` é usado para percorrer itens de uma sequência ou intervalo.
- A função `range()` gera uma sequência de números inteiros.

## Comando `for`
- Usado quando queremos iterar sobre um objeto iterável.
- Também é útil quando sabemos quantas vezes o bloco deve ser executado.

## Exemplo com texto
```python
texto = input("Informe um texto: ")
VOGAIS = "AEIOU"

for letra in texto:
    if letra.upper() in VOGAIS:
        print(letra, end="")
else:
    print()  # adiciona uma quebra de linha
```

## Observações sobre `for/else`
- O `else` do `for` é executado quando o laço termina normalmente.
- Nesse exemplo, ele é usado apenas para imprimir uma quebra de linha ao final.
- O `print(letra, end="")` evita quebrar linha a cada caractere exibido.

## Função `range()`
- Gera uma sequência de números inteiros.
- Forma geral:

```python
range(stop)
range(start, stop, step)
```

- `stop` é obrigatório.
- `start` e `step` são opcionais.
- O valor de `stop` não entra na sequência.

## Exemplo com `range`
```python
list(range(4))
# [0, 1, 2, 3]
```

## Exemplo com `for` e `range`
```python
for numero in range(0, 11):
    print(numero, end=" ")
```

```python
for numero in range(0, 51, 5):
    print(numero, end=" ")
```

## Observações importantes
- `range(0, 11)` gera de `0` até `10`.
- `range(0, 51, 5)` gera números de 5 em 5.
- O `for` pode iterar tanto em sequências quanto em intervalos numéricos.

# Estrutura de repetição (`while`)

## Resumo rápido
- O comando `while` repete um bloco de código **enquanto uma condição for verdadeira**.
- É ideal quando **não se sabe previamente** o número exato de repetições.
- Pode ser combinado com `else` para executar um bloco ao final da repetição (se não houver `break`).

---

## 1. O comando `while`

- Executa um bloco de código repetidamente enquanto a expressão lógica da condição for `True`.
- É fundamental que a condição se torne `False` em algum momento para evitar um loop infinito.

### Sintaxe básica

```python
while condicao:
    # Bloco de código a ser repetido
    # (É preciso alterar algo aqui para que a condição mude e o loop termine)
```

### Exemplo

```python
x = 1
while x <= 9:
    print(x)
    x = x + 1 # Incrementa x para que a condição eventualmente se torne falsa
```

---

## 2. Loop infinito (`while True`)

- Um `while True` cria um loop que, por si só, nunca termina.
- É usado quando se espera que o loop seja interrompido por uma condição interna, geralmente com `break`.

### Exemplo

```python
opcao = -1

while True: # Loop infinito
    opcao = int(input("[1] Sacar \n[0] Sair \n: "))

    if opcao == 1:
        print("Sacando...")
    elif opcao == 0:
        print("Saindo do sistema...")
        break # Interrompe o loop quando a opção for 0
    else:
        print("Opção inválida!")
```

---

## 3. `while/else`

- O bloco `else` associado a um `while` é executado **apenas se o loop terminar normalmente**, ou seja, quando a condição do `while` se torna `False`.
- Se o loop for interrompido por um `break`, o `else` **não é executado**.

### Sintaxe

```python
while condicao:
    # Bloco de código do loop
else:
    # Bloco de código executado se o loop terminar sem 'break'
```

### Exemplo

```python
opcao = -1

while opcao != 0:
    opcao = int(input("[1] Sacar \n[2] Extrato \n[0] Sair \n: "))

    if opcao == 1:
        print("Sacando...")
    elif opcao == 2:
        print("Exibindo o extrato...")
else:
    print("Obrigado por usar nosso sistema bancário, até logo!")
```

---

## 4. Palavras-chave de controle de fluxo (`break` e `continue`)

- Embora já abordadas em outra nota, é importante lembrar que `break` e `continue` são frequentemente usados dentro de loops `while` para controlar seu comportamento.

### `break`
- **Para** o loop imediatamente. O `else` do `while` **não é executado**.

### `continue`
- **Pula** a iteração atual e vai para a próxima.

---

## 5. Boas práticas

- Sempre garanta que a condição do `while` eventualmente se tornará `False` para evitar loops infinitos (a menos que seja intencional, com um `break`).
- Use `while` quando o número de repetições é desconhecido.
- Use `for` quando o número de repetições é conhecido ou quando se itera sobre uma sequência.
- O `else` do `while` é útil para executar código de "finalização" que só deve acontecer se o loop não foi interrompido abruptamente.

---

## 6. Resumo final

- `while` repete um bloco de código enquanto uma condição é `True`.
- `while True` cria um loop infinito, que deve ser controlado por um `break`.
- O `else` do `while` é executado se o loop terminar naturalmente (sem `break`).
- `break` e `continue` controlam o fluxo interno do loop.

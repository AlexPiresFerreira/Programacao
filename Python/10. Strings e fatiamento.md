
## Resumo rápido
- A classe `str` do Python é rica em métodos e fácil de trabalhar.
- Permite manipular textos com maiúsculas, minúsculas, espaços, junções, interpolações e fatiamentos.

---

## Métodos úteis da classe String

### Maiúsculas, minúsculas e título

```python
curso = "pYtHon"

print(curso.upper())   # PYTHON
print(curso.lower())   # python
print(curso.title())   # Python
```

### Removendo espaços em branco

```python
curso = " Python "

print(curso.strip())    # "Python"   → remove dos dois lados
print(curso.lstrip())   # "Python "  → remove da esquerda
print(curso.rstrip())   # " Python"  → remove da direita
```

### Centralização e junção

```python
curso = "Python"

print(curso.center(10, "#"))  # ##Python##
print(".".join(curso))         # P.y.t.h.o.n
```

## Interpolação de variáveis

Em Python existem 3 formas de interpolar variáveis em strings.

### Old style `%` *(não recomendado em Python 3)*

```python
nome = "Guilherme"
idade = 28

print("Me chamo %s e tenho %d anos." % (nome, idade))
```

### Método `.format()`

```python
nome = "Guilherme"
idade = 28
profissao = "Programador"
linguagem = "Python"

# por posição
print("Olá, me chamo {}. Tenho {} anos.".format(nome, idade))

# por índice
print("{3} tem {2} anos e usa {0}.".format(linguagem, profissao, idade, nome))

# por nome
print("{nome} trabalha como {profissao}.".format(nome=nome, profissao=profissao))
```

### f-string *(forma recomendada)*

```python
nome = "Guilherme"
idade = 28
profissao = "Programador"
linguagem = "Python"

print(f"Olá, me chamo {nome}. Tenho {idade} anos, trabalho como {profissao} e uso {linguagem}.")
```

### Formatando números com f-string

```python
PI = 3.14159

print(f"Valor de PI: {PI:.2f}")        # 3.14        → 2 casas decimais
print(f"Valor de PI: {PI:10.2f}")      # "      3.14" → 10 espaços + 2 casas decimais
```

**Sintaxe de formatação numérica:**

- `{variavel:.2f}` → define a quantidade de casas decimais
- `{variavel:10.2f}` → define o espaço total ocupado + casas decimais

## Fatiamento de string

- Técnica para retornar partes de uma string.
- Sintaxe: `string[start:stop:step]`
  - `start` → posição inicial (inclusiva)
  - `stop` → posição final (exclusiva)
  - `step` → passo (opcional)

```python
nome = "Guilherme Arthur de Carvalho"

print(nome[0])         # "G"
print(nome[:9])        # "Guilherme"
print(nome[10:])       # "Arthur de Carvalho"
print(nome[10:16])     # "Arthur"
print(nome[10:16:2])   # "Atu"
print(nome[:])         # "Guilherme Arthur de Carvalho"
print(nome[::-1])      # "ohlavraC ed ruhtrA emrehliuG" → string invertida
```

### Observações
- Índices negativos percorrem a string de trás para frente.
- `[::-1]` é o modo mais comum de inverter uma string.

## Strings de múltiplas linhas

- Definidas com três aspas simples `'''` ou duplas `"""`.
- Todos os espaços e quebras de linha são incluídos na string.

```python
nome = "Guilherme"

mensagem = f"""
Olá, meu nome é {nome}.
Eu estou aprendendo Python.
"""

print(mensagem)
```

```python
mensagem = f'''
    Olá meu nome é {nome},
    Eu estou aprendendo Python.
    Essa mensagem tem diferentes recuos.
'''

print(mensagem)
```

---

## Erros comuns
- Esquecer que índices em Python começam em `0`.
- Confundir `stop` com índice inclusivo — ele é **exclusivo**.
- Usar `%` para interpolar variáveis em Python 3 (prefira f-string).

## Boas práticas
- Prefira **f-string** para interpolar variáveis.
- Use `.strip()` ao receber entradas do usuário com `input()`.
- Use fatiamento para manipular partes de textos sem criar variáveis extras.

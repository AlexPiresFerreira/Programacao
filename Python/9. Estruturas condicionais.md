
## Resumo rápido
- Permitem o desvio do fluxo de controle quando determinadas expressões lógicas são atendidas.
- Python possui: `if`, `if/else`, `elif`, `if` aninhado e `if` ternário.

---

## `if` simples
- Testa uma expressão lógica.
- Executa o bloco apenas se o resultado for `True`.

```python
saldo = 2000.0
saque = float(input("Informe o valor do saque: "))

if saldo >= saque:
    print("Realizando saque!")

if saldo < saque:
    print("Saldo insuficiente!")
```

---

## `if / else`
- Dois desvios possíveis.
- Se a condição for `True`, executa o `if`.
- Se for `False`, executa o `else`.

```python
saldo = 2000.0
saque = float(input("Informe o valor do saque: "))

if saldo >= saque:
    print("Realizando saque!")
else:
    print("Saldo insuficiente!")
```

---

## `if / elif / else`
- Usado quando há mais de dois desvios possíveis.
- Cada `elif` testa uma nova condição.
- Não há limite de `elif`, mas evite estruturas muito longas.

```python
opcao = int(input("Informe uma opção: [1] Sacar \n[2] Extrato: "))

if opcao == 1:
    valor = float(input("Informe a quantia para o saque: "))
elif opcao == 2:
    print("Exibindo o extrato...")
else:
    print("Opção inválida")
```

---

## `if` aninhado
- Estruturas condicionais dentro de outras estruturas condicionais.
- Usado quando uma condição depende de outra.

```python
if conta_normal:
    if saldo >= saque:
        print("Saque realizado com sucesso!")
    elif saque <= (saldo + cheque_especial):
        print("Saque realizado com uso do cheque especial!")
elif conta_universitaria:
    if saldo >= saque:
        print("Saque realizado com sucesso!")
    else:
        print("Saldo insuficiente!")
```

---

## `if` ternário
- Escreve uma condição em uma única linha.
- Composto por três partes:
  1. Retorno se `True`
  2. Expressão lógica
  3. Retorno se `False`

```python
status = "Sucesso" if saldo >= saque else "Falha"
print(f"{status} ao realizar o saque!")
```

---

## Pontos importantes
- `if` testa uma condição.
- `else` é o caso contrário.
- `elif` adiciona novas condições intermediárias.
- `if` aninhado cria condições dentro de condições.
- `if` ternário é ideal para expressões simples de uma linha.

## Erros comuns
- Esquecer os dois pontos `:` após `if`, `elif` ou `else`.
- Não indentar corretamente o bloco de código.
- Usar `=` ao invés de `==` para comparar.

## If ternário

#### Resumo rápido
- O `if ternário` permite escrever uma condição em uma única linha.

#### Estrutura
- Forma geral:

```python
valor_se_verdadeiro if condicao else valor_se_falso
```

#### Exemplo
```python
saldo = 2000
saque = 2500

status = "Sucesso" if saldo >= saque else "Falha"
print(f"{status} ao realizar o saque!")
```

#### Observações
- É uma forma compacta de `if / else`.
- Deve ser usada quando a lógica for simples.
- Se a condição ficar complexa, prefira a estrutura tradicional com blocos.

## 1. `if` (Se)

- É a estrutura condicional mais básica.
- Executa um bloco de código **apenas se** a condição for verdadeira.

### Sintaxe

```python
if condicao:
    # Bloco de código executado se a condição for True
```

### Exemplo

```python
saldo = 500
saque = 200

if saldo >= saque:
    print("Saque realizado com sucesso!")
```

---

## 2. `elif` (Se não, se)

- Usado para testar **condições adicionais** se o `if` anterior for falso.
- Você pode ter múltiplos `elif`s.

### Sintaxe

```python
if condicao1:
    # Bloco de código se condicao1 for True
elif condicao2:
    # Bloco de código se condicao1 for False E condicao2 for True
```

### Exemplo

```python
opcao = 1

if opcao == 1:
    print("Você escolheu Sacar.")
elif opcao == 2:
    print("Você escolheu Extrato.")
```

---

## 3. `else` (Caso contrário)

- Executa um bloco de código se **nenhuma** das condições anteriores (`if` ou `elif`) for verdadeira.
- É opcional e só pode haver um `else` por estrutura condicional.

### Sintaxe

```python
if condicao1:
    # Bloco de código se condicao1 for True
elif condicao2:
    # Bloco de código se condicao1 for False E condicao2 for True
else:
    # Bloco de código se todas as condições anteriores forem False
```

### Exemplo

```python
opcao = 3

if opcao == 1:
    print("Você escolheu Sacar.")
elif opcao == 2:
    print("Você escolheu Extrato.")
else:
    print("Opção inválida. Tente novamente.")
```

---

## 4. `if` com `break` e `continue` (em loops)

- Embora `if` seja uma estrutura condicional, ele é frequentemente usado **dentro de loops** (`for` ou `while`) para controlar o fluxo do loop com `break` e `continue`.

### `break`
- Usado dentro de um `if` para **parar completamente** o loop.

```python
for numero in range(5):
    if numero == 3:
        break  # O loop para quando numero for 3
    print(numero)
# Saída: 0, 1, 2
```

### `continue`
- Usado dentro de um `if` para **pular a iteração atual** do loop e ir para a próxima.

```python
for numero in range(5):
    if numero == 2:
        continue  # Pula a impressão quando numero for 2
    print(numero)
# Saída: 0, 1, 3, 4
```

---

## 5. Palavras-chave e seus significados

- `if` == **se** (testa uma condição)
- `elif` == **se não, se** (testa uma condição alternativa)
- `else` == **caso contrário** (executa se nenhuma condição anterior for verdadeira)
- `break` == **para o loop** (interrompe a repetição)
- `continue` == **executa a próxima iteração** (pula o restante da iteração atual)

---

## 6. Boas práticas

- Use `if/elif/else` para tomar decisões claras no seu código.
- Evite aninhar muitos `if`s (um `if` dentro de outro `if`), pois isso pode dificultar a leitura. Considere usar `elif` ou refatorar a lógica.
- Use `break` e `continue` com moderação para manter o fluxo do loop fácil de entender.

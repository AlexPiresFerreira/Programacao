
```python
import requests
from bs4 import BeautifulSoup

def listar_repositorios(username):
    url = f"https://github.com/{username}?tab=repositories"
    resposta = requests.get(url)

    if resposta.status_code != 200:
        print("Erro ao acessar o GitHub")
        return

    sopa = BeautifulSoup(resposta.text, "html.parser")
    repos = sopa.select('h3 a[itemprop="name codeRepository"]')

    for repo in repos:
        print(repo.text.strip())
```

### O que esse exemplo faz

- Acessa a página de repositórios de um usuário no GitHub.
- Faz o parsing do HTML.
- Lista os nomes dos repositórios encontrados.

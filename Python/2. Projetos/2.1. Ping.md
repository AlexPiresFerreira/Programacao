
- Ping simples

```
import os

ip_ou_host = input("Digite o IP ou HOST a ser verificado: ")

os.system(f'ping -n 6 {ip_ou_host}')
```

- Ping multiplo 

```
import os 
import time

# arquivo de texto 
with open('[arquivo.txt]') as file:
	[variavel] = file.read()
	[variavel] = [variavel].splitlines()
	
	for ip in [variavel]:
		os.system(f'ping -n 2 {ip}')
		time.sleep(5)

```

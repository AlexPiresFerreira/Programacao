
### Python — servidor TCP com TLS e autenticação mútua

```python
#!/usr/bin/python3
import socket
from socket import AF_INET, SOCK_STREAM, SO_REUSEADDR, SOL_SOCKET, SHUT_RDWR
import ssl

listen_addr = '127.0.0.1'
listen_port = 9001
server_cert = 'server.crt'
server_key = 'server.key'
client_certs = 'client.crt'

context = ssl.create_default_context(ssl.Purpose.CLIENT_AUTH)
context.verify_mode = ssl.CERT_REQUIRED
context.load_cert_chain(certfile=server_cert, keyfile=server_key)
context.load_verify_locations(cafile=client_certs)

sock = socket.socket()
sock.bind((listen_addr, listen_port))
sock.listen(5)

while True:
    print("Aguardando clientes")
    (csock, caddr) = sock.accept()
    print(f"cliente: {caddr[0]}:{caddr[1]}")
    ssock = context.wrap_socket(csock, server_side=True)
    buf = b' '
    try:
        while True:
            data = ssock.recv(4096)
            if data:
                buf += data
            else:
                print("Received:", buf)
                break
    finally:
        print("O cliente encerrou")
        ssock.close()
```

**Pontos-chave para resumo:**
- Usa `ssl.create_default_context(ssl.Purpose.CLIENT_AUTH)` para exigir **certificado de cliente**.
- `verify_mode = ssl.CERT_REQUIRED` → conexão só é aceita se o cliente apresentar um certificado válido.
- `load_cert_chain` → carrega o par chave/certificado do servidor.
- `load_verify_locations` → indica a CA/arquivo de certificados aceitos para clientes.
- É um exemplo de **autenticação mútua (mTLS)** em Python.

### Python — cliente TCP com TLS e certificado de cliente

```python
#!/usr/bin/python3
import socket
import ssl

host_addr = '127.0.0.1'
host_port = 9001
hostname = 'www.4bios.com.br'
server_cert = 'server.crt'
client_cert = 'client.crt'
client_key = 'client.key'

context = ssl.create_default_context(
    ssl.Purpose.SERVER_AUTH,
    cafile=server_cert
)
context.load_cert_chain(certfile=client_cert, keyfile=client_key)

sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
ssock = context.wrap_socket(sock, server_side=False, server_hostname=hostname)
ssock.connect((host_addr, host_port))

ssock.send(b"Mensagem do cliente, Ola Mundo...rsrsrsrsrs")

ssock.close()
```

**Ideias importantes:**
- `ssl.Purpose.SERVER_AUTH` → valida o certificado do servidor.
- `cafile=server_cert` → arquivo usado para confiar no servidor.
- `server_hostname=hostname` → utilizado para SNI e checagem do nome no certificado.

### Python — inspecionando certificado SSL de um servidor

**Ideias principais dos scripts do material:**

- Abrir uma conexão TLS com `ssl` e pegar o certificado apresentado pelo servidor:
  - `cert = ssock.getpeercert()` → retorna o certificado em forma de dicionário.
- Validar se o **hostname bate com o certificado**:
  - `ssl.match_hostname(cert, hostname)` → lança exceção se não confere.
- Converter o certificado para uma estrutura mais legível:
  - Percorrer `cert["subject"]` e montar um dicionário `server[...]` com campos como `commonName`.
- Usos práticos:
  - Confirmar que você está falando com o servidor certo.
  - Ver detalhes como emissor, validade e CN do certificado.


# Python — Programação Orientada a Objetos (OOP)

## Conceitos básicos

- **Classe:** Um "molde" ou "planta" para criar objetos. Define características (atributos) e comportamentos (métodos).
- **Objeto (Instância):** Uma ocorrência concreta de uma classe.
- **Atributo:** Uma característica do objeto (variável dentro da classe).
- **Método:** Uma ação ou comportamento que o objeto pode realizar (função dentro da classe).
- `self`: Referência ao próprio objeto dentro da classe. É o primeiro parâmetro de todos os métodos de instância.

## Definindo uma classe

```python
class Veiculo: # Nome da classe (convenção: CamelCase)
    def __init__(self, cor, rodas, marca): # Método construtor
        self.cor = cor     # Atributo de instância
        self.rodas = rodas # Atributo de instância
        self.marca = marca # Atributo de instância
        self.tanque = 0    # Atributo com valor inicial

    def abastecer(self, litros): # Método de instância
        if self.tanque + litros > 50:
            print("Erro: tanque com capacidade inferior a 50 litros.")
        else:
            self.tanque += litros
            print(f"Abastecido com {litros} litros. Tanque atual: {self.tanque} litros.")
```

## Criando e usando objetos

```python
# Criando uma instância (objeto) da classe Veiculo
caminhao_rosa = Veiculo('rosa', 6, 'Ford')

print("Cor:", caminhao_rosa.cor)
print("Marca:", caminhao_rosa.marca)
print("Tanque inicial:", caminhao_rosa.tanque)

caminhao_rosa.abastecer(35)
print("Tanque após abastecer:", caminhao_rosa.tanque)

caminhao_rosa.abastecer(20) # Tentativa de exceder o tanque
```

## Herança (novo)

- Permite que uma classe (subclasse) herde atributos e métodos de outra classe (superclasse).
- Promove a reutilização de código.

```python
# Classe Carro herda de Veiculo
class Carro(Veiculo):
    def __init__(self, cor, rodas, marca, tanque, portas):
        # Chama o construtor da classe pai (Veiculo)
        super().__init__(cor, rodas, marca)
        self.tanque = tanque # Pode sobrescrever ou adicionar atributos
        self.portas = portas # Novo atributo específico de Carro

    def ligar_radio(self):
        print("Rádio ligado!")

# Criando um objeto da subclasse Carro
meu_carro = Carro('azul', 4, 'Chevrolet', 40, 4)
print("Cor do carro:", meu_carro.cor)
meu_carro.abastecer(10) # Método herdado de Veiculo
meu_carro.ligar_radio() # Método específico de Carro
```


### Biblioteca `json`

- Usada para **codificar e decodificar dados JSON** (JavaScript Object Notation).
- Essencial para trabalhar com APIs web.

```python
import requests
import json

# Exemplo de requisição a uma API que retorna JSON
req = requests.get("http://www.omdbapi.com/?t=interstellar&apikey=SEU_API_KEY") # Substitua SEU_API_KEY

# O conteúdo da resposta é uma string JSON
json_string = req.text
print("JSON recebido (string):", json_string)

# json.loads: converte string JSON para dicionário Python
dicionario_python = json.loads(json_string)
print("JSON convertido (dicionário Python):", dicionario_python)

# Acessando dados do dicionário
print("Título do filme:", dicionario_python['Title'])
```

**Pontos-chave:**
- `json.loads(string_json)`: Converte uma string JSON em um objeto Python (geralmente um dicionário ou lista).
- `json.dumps(objeto_python)`: Converte um objeto Python em uma string JSON.****

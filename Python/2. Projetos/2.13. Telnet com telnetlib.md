
### Python — automação básica com Telnet

```python
from telnetlib import Telnet

with Telnet('172.16.25.186', 23) as tn:
    tn.interact()
```

- `Telnet(host, porta)` → abre uma sessão Telnet.
- `tn.interact()` → entrega o controle do terminal para você, como se estivesse num cliente Telnet normal.
- Útil para **testes rápidos em equipamentos legados** que ainda usam Telnet.


### Biblioteca `requests`

- Usada para fazer **requisições HTTP** (GET, POST, etc.) de forma simples.
- Facilita a interação com APIs e serviços web.

```python
import requests

# Requisição GET
requisicao_get = requests.get("https://solyd.com.br")
print(requisicao_get.text)

# Requisição POST com cabeçalhos, cookies e dados
cabecalho = {"User-agent": "Windows 12"}
meus_cookies = {"Ultima-visita": "10-10-2020"}
dados = {"username": "guigui", "password": "guigui123"}

requisicao_post = requests.post("https://solyd.com.br", headers=cabecalho, cookies=meus_cookies, data=dados)
# ... processar requisicao_post
```

**Pontos-chave:**
- `requests.get(url)`: Faz uma requisição GET.
- `requests.post(url, headers=..., cookies=..., data=...)`: Faz uma requisição POST com opções.
- `requisicao.text`: Conteúdo da resposta como string.


- Biblioteca fundamental para **comunicação em rede**
- Serve para criar clientes e servidores TCP/UDP
- No documento, aparece em exemplos de:
    - resolução de hostname
    - ping
    - conexão entre máquinas
    - servidor e cliente

### Resolver nome para IP


```python
socket.gethostbyname("example.com")
```

- Descobre o endereço IP associado a um domínio.

### Resolver IP para nome


```python
socket.gethostbyaddr("8.8.8.8")
```

- Faz o caminho inverso: IP → host.

### Uso prático

- Scripts de rede
- Resolução DNS
- Automação de verificações

### Importar um módulo inteiro


```python
import time
```

- Importa o módulo completo.
- Para usar, acessa-se com o nome do módulo:

```python
time.sleep(1)
```

### Importar uma função específica


```python
from os import system
```

- Importa apenas o que foi pedido.
- Torna o uso mais direto:

```python
system("clear")
```

### Dar apelido ao import

```python
import requests as r
```

- Cria um alias para facilitar a escrita.

```python
r.get("https://example.com")
```

### `__init__.py`

- Indica que o diretório é um **pacote Python**.
- É uma boa prática para organizar projetos com múltiplos módulos.

### Biblioteca `re` (Expressões Regulares)

- Usada para **procurar padrões** em strings.
- Útil para validação de dados, extração de informações (ex: e-mails, telefones).

```python
import re

string_de_teste = "o gato é bonito"

# re.search: procura a primeira ocorrência do padrão
padrao_search = re.search(r"gato", string_de_teste)
if padrao_search:
    print("Padrão encontrado:", padrao_search.group()) # 'gato'

# re.findall: procura todas as ocorrências do padrão
padrao_findall = re.findall(r"\w\w\w\w", string_de_teste) # Procura sequências de 4 letras/números
print("Todas as ocorrências:", padrao_findall) # ['gato', 'boni']

# Modificadores de quantidade:
# \w+ : uma ou mais ocorrências de caracteres alfanuméricos
# \w* : zero ou mais ocorrências de caracteres alfanuméricos
```

**Pontos-chave:**
- `r"..."`: String "raw" (bruta), para evitar problemas com `\` em regex.
- `re.search(padrao, texto)`: Retorna um objeto `Match` ou `None`.
- `re.findall(padrao, texto)`: Retorna uma lista de todas as correspondências.
- `\w`: Corresponde a caracteres alfanuméricos (letras, números, underscore).



```python
import re

padrao = re.compile(r"\d+")
resultado = padrao.findall("Tenho 2 gatos e 3 cachorros")
print(resultado)  # ['2', '3']
```

### Funções importantes

- `re.compile()`
- `findall()`
- `search()`
- `match()`
- `sub()`

### Observação

- Regex é muito útil para validação, extração e busca de padrões em texto.

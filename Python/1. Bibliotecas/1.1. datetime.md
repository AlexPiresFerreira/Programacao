
# Data, hora e fuso horário

## Resumo rápido
- O módulo `datetime` é usado para trabalhar com **datas, horas, diferenças entre datas e fusos horários**.
- Ele oferece classes úteis como `date`, `time`, `datetime` e `timedelta`.
- Também permite **formatar** e **converter** datas e horas com `strftime()` e `strptime()`.

---

## O módulo `datetime`

O módulo `datetime` é a base para manipular datas e horas em Python.

### Principais classes
- `date` → representa uma data
- `time` → representa uma hora
- `datetime` → representa data e hora juntas
- `timedelta` → representa diferença entre datas/horas

### Exemplo básico

```python
from datetime import date, datetime

data = date(2023, 7, 10)
print(data)

data_hora = datetime(2023, 7, 10)
print(data_hora)

print(datetime.today())
```

---

## Manipulação de datas e horas

Com `datetime`, podemos:
- criar datas e horas;
- somar ou subtrair períodos;
- comparar datas;
- descobrir diferenças entre dois momentos.

### Exemplo com `timedelta`

```python
from datetime import datetime, timedelta

data_hora = datetime(2023, 7, 10, 15, 30)
print(data_hora)

nova_data = data_hora + timedelta(days=7)
print(nova_data)

dif = nova_data - data_hora
print(dif)
```

### Observações
- `timedelta` é muito útil para lidar com intervalos de tempo.
- Você pode alterar dias, horas, minutos e segundos.

---

## Conversão e formatação de datas e horas

Python permite transformar datas em texto e texto em datas.

### `strftime()`
- Converte um objeto de data/hora em **string formatada**.

```python
from datetime import datetime

agora = datetime.now()

print(agora.strftime("%d/%m/%Y"))
print(agora.strftime("%H:%M:%S"))
print(agora.strftime("%d/%m/%Y %H:%M:%S"))
```

### `strptime()`
- Converte uma **string** em objeto `datetime`.

```python
from datetime import datetime

texto = "10/07/2023 15:30:00"
data_hora = datetime.strptime(texto, "%d/%m/%Y %H:%M:%S")

print(data_hora)
```

### Observações importantes
- `strftime()` → **datetime para string**
- `strptime()` → **string para datetime**
- Os formatos precisam bater exatamente com o texto informado.

---

## Formatação mais comum

### Códigos úteis de formatação
- `%d` → dia com dois dígitos
- `%m` → mês com dois dígitos
- `%Y` → ano com quatro dígitos
- `%H` → hora em formato 24h
- `%M` → minutos
- `%S` → segundos

### Exemplo

```python
from datetime import datetime

agora = datetime.now()

print(agora.strftime("%d/%m/%Y"))
print(agora.strftime("%d/%m/%Y %H:%M:%S"))
```

## Trabalhando com fusos horários

Quando lidamos com data e hora, o fuso horário é muito importante, principalmente em aplicações que:
- atendem usuários de diferentes regiões;
- registram eventos em horários distintos;
- precisam manter consistência entre sistemas.

### Usando bibliotecas externas
- O material cita o uso de `pytz` para facilitar o trabalho com timezones.

```python
from datetime import datetime
import pytz

agora = datetime.now(pytz.timezone("America/Sao_Paulo"))
print(agora)
```

### Observações
- Fusos horários dependem da base de dados de timezones.
- O material também aponta uma lista de timezones baseada na base IANA.

---

## Trabalhando com timezone sem bibliotecas externas

O Python também permite lidar com fuso horário usando o próprio `datetime`, embora a abordagem seja mais trabalhosa.

### Exemplo conceitual

```python
from datetime import datetime, timezone, timedelta

fuso_sp = timezone(timedelta(hours=-3))
agora_sp = datetime.now(fuso_sp)

print(agora_sp)
```

### Observações
- Sem bibliotecas externas, o controle manual é possível.
- Em muitos casos, usar uma biblioteca especializada facilita bastante.

---

## Boas práticas
- Use `datetime` para manipular datas e horas de forma clara.
- Use `timedelta` para somar ou subtrair períodos.
- Use `strftime()` e `strptime()` para conversão entre texto e data.
- Preste atenção ao fuso horário quando os dados dependerem de localidade.

---

## Erros comuns
- Confundir `strftime()` com `strptime()`.
- Esquecer que o formato da string precisa corresponder ao padrão informado.
- Trabalhar com datas sem considerar o fuso horário quando isso for importante.
- Usar `datetime.today()` sem entender se o horário local é o desejado.


## Resumo rápido
- Listas armazenam sequencialmente qualquer tipo de objeto.
- São mutáveis, ou seja, seus valores podem ser alterados após a criação.

---

## Criando listas

```python
frutas = ["laranja", "maçã", "uva"]   # lista com valores
vazia = []                             # lista vazia
letras = list("python")               # cada letra vira um item → ['p','y','t','h','o','n']
numeros = list(range(10))             # lista de 0 a 9
carro = ["Ferrari", "F8", 4200000, 2020, True]  # tipos mistos
```

---

## Acesso direto

- Índices começam em **0**.
- Índices negativos percorrem de **trás para frente**, começando em **-1**.

```python
frutas = ["maçã", "laranja", "uva", "pera"]

frutas[0]    # "maçã"
frutas[2]    # "uva"
frutas[-1]   # "pera"
frutas[-3]   # "laranja"
```

---

## Listas aninhadas

- Listas podem armazenar outras listas.
- Acesso feito com dois índices: `[linha][coluna]`.

```python
matriz = [
    [1, "a", 2],
    ["b", 3, 4],
    [6, 5, "c"]
]

matriz[0]       # [1, "a", 2]
matriz[0][0]    # 1
matriz[0][-1]   # 2
matriz[-1][-1]  # "c"
```

---

## Fatiamento

- Extrai partes da lista sem modificar a original.
- Sintaxe: `lista[start:stop:step]`

```python
lista = ["p", "y", "t", "h", "o", "n"]

lista[2:]     # ["t", "h", "o", "n"]
lista[:2]     # ["p", "y"]
lista[1:3]    # ["y", "t"]
lista[0:3:2]  # ["p", "t"]
lista[::]     # ["p", "y", "t", "h", "o", "n"]
lista[::-1]   # ["n", "o", "h", "t", "y", "p"] → lista invertida
```

---

## Iterando listas

### Com `for`
```python
carros = ["gol", "celta", "palio"]

for carro in carros:
    print(carro)
```

### Com `enumerate` (quando precisar do índice)
```python
for indice, carro in enumerate(carros):
    print(f"{indice}: {carro}")
```

---

## Compreensão de listas

Forma mais enxuta de criar listas com filtros ou modificações.

### Filtro
```python
numeros = [1, 30, 21, 2, 9, 65, 34]

# forma tradicional
pares = []
for numero in numeros:
    if numero % 2 == 0:
        pares.append(numero)

# forma compacta
pares = [numero for numero in numeros if numero % 2 == 0]
```

### Modificando valores
```python
# forma tradicional
quadrado = []
for numero in numeros:
    quadrado.append(numero ** 2)

# forma compacta
quadrado = [numero ** 2 for numero in numeros]
```

---

## Métodos da classe list

| Método | Descrição |
|---|---|
| `.append(x)` | Adiciona um item ao final |
| `.clear()` | Remove todos os itens |
| `.copy()` | Retorna uma cópia da lista |
| `.count(x)` | Conta quantas vezes o valor aparece |
| `.extend([x])` | Adiciona múltiplos itens ao final |
| `.index(x)` | Retorna o índice do valor |
| `.pop(i)` | Remove e retorna o item pelo índice |
| `.remove(x)` | Remove a primeira ocorrência do valor |
| `.reverse()` | Inverte a lista |
| `.sort()` | Ordena a lista |

### Exemplos

```python
lista = []
lista.append(1)
lista.append("Python")
lista.append([40, 30, 20])
print(lista)  # [1, "Python", [40, 30, 20]]

lista.clear()
print(lista)  # []
```

```python
cores = ["vermelho", "azul", "verde", "azul"]
cores.count("azul")   # 2
cores.count("verde")  # 1
```

```python
linguagens = ["python", "js", "c"]
linguagens.extend(["java", "csharp"])
print(linguagens)  # ["python", "js", "c", "java", "csharp"]
```

```python
linguagens = ["python", "js", "c", "java", "csharp"]
linguagens.pop()    # remove "csharp"
linguagens.pop(0)   # remove "python"
linguagens.remove("c")
print(linguagens)  # ["js", "java"]
```

```python
linguagens = ["python", "js", "c", "java", "csharp"]
linguagens.reverse()
print(linguagens)  # ["csharp", "java", "c", "js", "python"]
```

```python
linguagens = ["python", "js", "c", "java", "csharp"]
linguagens.sort()                              # ordem alfabética
linguagens.sort(reverse=True)                  # ordem inversa
linguagens.sort(key=lambda x: len(x))         # por tamanho
linguagens.sort(key=lambda x: len(x), reverse=True)  # por tamanho inverso
```

---

## Funções úteis com listas

```python
linguagens = ["python", "js", "c", "java", "csharp"]

len(linguagens)  # 5 → quantidade de itens

sorted(linguagens, key=lambda x: len(x))               # não modifica a original
sorted(linguagens, key=lambda x: len(x), reverse=True)
```

### Diferença entre `sort()` e `sorted()`
- `.sort()` → modifica a lista original.
- `sorted()` → retorna uma nova lista, sem alterar a original.

---

## Erros comuns
- Acessar um índice que não existe gera `IndexError`.
- Confundir `.append()` com `.extend()`:
  - `.append([1,2])` → adiciona a lista como um único item.
  - `.extend([1,2])` → adiciona cada item separadamente.

## Boas práticas
- Prefira compreensão de lista para filtros e transformações simples.
- Use `enumerate()` quando precisar do índice no `for`.
- Use `sorted()` quando não quiser alterar a lista original.

# Python - Listas (complemento e revisão)

## Resumo rápido
- Listas são coleções ordenadas e **mutáveis** de itens.
- Permitem armazenar diferentes tipos de dados.
- O acesso aos elementos é feito por índice, começando em `0`.

---

## 1. Criação e acesso básico

### Criando listas
```python
variavel = [valor1, valor2, valor3]  # Lista com valores
variavel = []                        # Lista vazia
variavel = list('texto')             # Cada letra vira um item: ['t', 'e', 'x', 't', 'o']
variavel = list(range(10))           # Cria uma lista de 0 a 9
```

### Acesso por índice
- O primeiro elemento está no índice `0`.
- Índices negativos acessam de trás para frente.

```python
lista = ["indexe0", "indexe1", "indexe2"]

print(lista[0])   # "indexe0"
print(lista[1])   # "indexe1"
print(lista[-1])  # "indexe2" (último elemento)
print(lista[-2])  # "indexe1" (penúltimo elemento)
```

---

## 2. Modificando listas

### Trocar um valor
- Acesse o elemento pelo índice e atribua um novo valor.

```python
lista = ["a", "b", "c"]
lista[1] = "novo_b"
print(lista)  # ["a", "novo_b", "c"]
```

### Copiar um valor de uma referência para outra
- Atribuição direta de um elemento para outro.

```python
lista = [10, 20, 30]
lista[0] = lista[2]
print(lista)  # [30, 20, 30]
```

### Criar uma cópia independente (fatiamento completo)
- Usa fatiamento `[:]` para criar uma nova lista, não apenas uma referência.

```python
lista1 = [1, 2, 3]
lista2 = lista1[:]  # Cria uma cópia independente
lista2[0] = 99
print(lista1)  # [1, 2, 3]
print(lista2)  # [99, 2, 3]
```

---

## 3. Fatiamento (Slicing)

- Extrai uma sub-lista da lista original.
- Sintaxe: `lista[valor_inicial : valor_final]`
- O `valor_final` **não é incluído** no fatiamento.

```python
lista = ["a", "b", "c", "d", "e"]

print(lista[1:4])  # ["b", "c", "d"]
print(lista[:3])   # ["a", "b", "c"]
print(lista[2:])   # ["c", "d", "e"]
```

---

## 4. Métodos para adicionar valores

### `append()`
- Adiciona um elemento ao **final** da lista.

```python
lista = [1, 2]
lista.append(3)
print(lista)  # [1, 2, 3]
```

### `insert()`
- Adiciona um elemento em um **índice específico**.

```python
lista = [1, 3]
lista.insert(1, 2)  # Adiciona 2 no índice 1
print(lista)  # [1, 2, 3]
```

---

## 5. Métodos para remover valores

### `pop()`
- Remove e retorna o **último** elemento da lista.
- Pode remover um elemento em um índice específico se o índice for passado como argumento.

```python
lista = [1, 2, 3]
ultimo = lista.pop()
print(lista)  # [1, 2]
print(ultimo) # 3

lista.pop(0)  # Remove o elemento no índice 0
print(lista)  # [2]
```

### `del`
- Deleta um elemento ou um fatiamento da lista.

```python
lista = [1, 2, 3, 4, 5]
del lista[0]        # Deleta o elemento no índice 0
print(lista)        # [2, 3, 4, 5]

del lista[1:3]      # Deleta do índice 1 ao 2 (o 3 não é incluído)
print(lista)        # [2, 5]
```

### `remove()`
- Remove a **primeira ocorrência** de um valor específico.

```python
lista = [1, 2, 3, 2]
lista.remove(2)
print(lista)  # [1, 3, 2]
```

---

## 6. Métodos para ordenar e contar

### `reverse()`
- Inverte a ordem dos elementos na lista (modifica a lista original).

```python
lista = [1, 2, 3]
lista.reverse()
print(lista)  # [3, 2, 1]
```

### `sort()`
- Ordena os elementos da lista (modifica a lista original).
- Por padrão, em ordem crescente (alfabética para strings).

```python
lista = [3, 1, 2]
lista.sort()
print(lista)  # [1, 2, 3]
```

### `count()`
- Conta a quantidade de vezes que um valor aparece na lista.

```python
lista = [1, 2, 2, 3, 1]
contagem = lista.count(2)
print(contagem)  # 2
```

---

## 7. Operadores de associação (`in`, `not in`)

- Verificam se um valor está presente ou não na lista.
- Retornam `True` ou `False`.

```python
lista = [10, 20, 30]

print(10 in lista)      # True
print(50 not in lista)  # True
```

---

## 8. Estruturas de dados com listas

### FIFO (First-In, First-Out) - Fila
- O primeiro elemento a entrar é o primeiro a sair.
- Pode ser implementado com `append()` e `pop(0)`.

### LIFO (Last-In, First-Out) - Pilha
- O último elemento a entrar é o primeiro a sair.
- Pode ser implementado com `append()` e `pop()`.

---

## 9. Resumo final

- Listas são mutáveis e flexíveis.
- Acesso por índice (`[]`), fatiamento (`[:]`).
- Métodos de modificação: `append()`, `insert()`, `pop()`, `remove()`, `del`.
- Métodos de ordenação/contagem: `reverse()`, `sort()`, `count()`.
- Operadores `in` e `not in` para verificar pertencimento.
- Podem ser usadas para implementar filas (FIFO) e pilhas (LIFO).

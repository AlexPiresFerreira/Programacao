
## Visão geral
Os operadores são símbolos usados para realizar cálculos, comparações, atribuições e verificações entre valores e objetos.

---

## Operadores aritméticos

Usados para realizar operações matemáticas.

- **Adição:** `+`
- **Subtração:** `-`
- **Multiplicação:** `*`
- **Divisão:** `/`
- **Divisão inteira:** `//`
- **Módulo:** `%`
- **Exponenciação:** `**`

### Exemplo
```python
print(1 + 1)    # 2
print(10 - 2)   # 8
print(4 * 3)    # 12
print(12 / 3)   # 4.0
print(12 // 2)  # 6
print(10 % 3)   # 1
print(2 ** 3)   # 8
```

### Ordem de execução
Quando uma expressão tem mais de uma operação, Python segue a ordem abaixo:

1. Parênteses
2. Expoentes
3. Multiplicações e divisões da esquerda para a direita
4. Somas e subtrações da esquerda para a direita

### Exemplo
```python
print(10 - 5 * 2)      # 0
print((10 - 5) * 2)    # 10
print(10 ** 2 * 2)     # 200
print(10 ** (2 * 2))   # 10000
print(10 / 2 * 4)      # 20.0
```

## Operadores de comparação

Usados para comparar dois valores e retornar `True` ou `False`.

- **Igualdade:** `==`
- **Diferença:** `!=`
- **Maior que:** `>`
- **Maior ou igual:** `>=`
- **Menor que:** `<`
- **Menor ou igual:** `<=`

### Exemplo
```python
saldo = 450
saque = 200

print(saldo == saque)  # False
print(saldo != saque)  # True
print(saldo > saque)   # True
print(saldo >= saque)  # True
print(saldo < saque)   # False
print(saldo <= saque)  # False
```

---

## Operadores de atribuição

Usados para definir ou atualizar o valor de uma variável.

- **Atribuição simples:** `=`
- **Adição:** `+=`
- **Subtração:** `-=`
- **Multiplicação:** `*=`
- **Divisão:** `/=`
- **Divisão inteira:** `//=`
- **Módulo:** `%=`
- **Exponenciação:** `**=`

### Exemplo
```python
saldo = 500

saldo += 200   # 700
saldo -= 100   # 600
saldo *= 2     # 1200
saldo /= 5     # 240.0
saldo //= 5    # 48.0
saldo %= 40    # 8.0
saldo **= 2    # 64.0
```

### Observação
- `=` faz atribuição.
- Os demais operadores já fazem a operação e atualizam a variável ao mesmo tempo.

## Operadores lógicos

Usados para combinar expressões booleanas.

- **E:** `and`
- **Ou:** `or`
- **Negação:** `not`

### Exemplo
```python
saldo = 1000
saque = 200
limite = 100

print(saldo >= saque and saque <= limite)  # False
print(saldo >= saque or saque <= limite)   # True
print(not 1000 > 1500)                     # True
```

### Uso de parênteses
Parênteses ajudam a deixar a expressão mais clara e controlar a prioridade lógica.

```python
saldo = 1000
saque = 250
limite = 200
conta_especial = True

print((saldo >= saque and saque <= limite) or (conta_especial and saldo >= saque))
```

## Operadores de identidade

Usados para verificar se dois objetos ocupam a mesma posição na memória.

- **É o mesmo objeto:** `is`
- **Não é o mesmo objeto:** `is not`

### Exemplo
```python
curso = "Curso de Python"
nome_curso = curso
saldo = 200
limite = 200

print(curso is nome_curso)       # True
print(curso is not nome_curso)   # False
print(saldo is limite)           # True
```

### Observação
- `is` compara identidade, não valor.
- Em muitos casos, o mais importante é saber se dois objetos são o mesmo objeto em memória.

---

## Operadores de associação

Usados para verificar se um elemento está dentro de uma sequência.

- **Pertence:** `in`
- **Não pertence:** `not in`

### Exemplo
```python
curso = "Curso de Python"
frutas = ["laranja", "uva", "limão"]
saques = [1500, 100]

print("Python" in curso)      # True
print("maçã" not in frutas)    # True
print(200 in saques)           # False
```

## Precedência dos operadores

A ordem de execução das operações segue a seguinte hierarquia (do maior para o menor):

1. `()` → parênteses sempre calculados primeiro
2. `**` → exponenciação
3. `+x`, `-x` → sinais unários (positivo/negativo)
4. `*`, `/`, `//`, `%` → multiplicação e divisão
5. `+`, `-` → adição e subtração (binários)
6. `<`, `<=`, `>`, `>=` → comparações de ordem
7. `==`, `!=` → comparações de igualdade

```python
print(2 + 3 * 4)        # 14  (multiplicação antes)
print((2 + 3) * 4)      # 20  (parêntese primeiro)
print(2 ** 3 + 1)       # 9   (exponenciação antes)
print(10 - 4 // 2)      # 8   (divisão inteira antes)
```

## Operadores de comparação

Retornam sempre `True` ou `False`.

| Operador | Significado |
|----------|------------|
| `==` | Igual |
| `!=` | Diferente |
| `>` | Maior que |
| `>=` | Maior ou igual |
| `<` | Menor que |
| `<=` | Menor ou igual |

```python
print(5 == 5)   # True
print(5 != 3)   # True
print(5 > 3)    # True
print(5 >= 5)   # True
print(3 < 5)    # True
print(3 <= 3)   # True
```

## Operadores lógicos

Combinam expressões booleanas.

| Operador | Significado |
|----------|------------|
| `and` | Verdadeiro se **ambas** as condições forem verdadeiras |
| `or` | Verdadeiro se **ao menos uma** condição for verdadeira |
| `not` | Inverte o resultado |

```python
print(True and False)   # False
print(True or False)    # True
print(not True)         # False

# combinando com comparações
x = 5
print(x > 3 and x < 10)   # True
print(x < 3 or x < 10)    # True
print(not x > 3)           # False
```

### Uso de parênteses em expressões lógicas
- Parênteses deixam a intenção mais clara e controlam a precedência.

```python
resultado = (True or False) and not False
print(resultado)  # True
```

## Operadores de identidade

Verificam se dois objetos ocupam a **mesma posição na memória**.

| Operador | Significado |
|----------|------------|
| `is` | É o mesmo objeto |
| `is not` | Não é o mesmo objeto |

```python
a = [1, 2, 3]
b = a
c = [1, 2, 3]

print(a is b)   # True  → mesmo objeto
print(a is c)   # False → objetos diferentes (mesmo valor)
print(a == c)   # True  → valores iguais
```

> **Atenção:** use `is` para verificar identidade.  
> Use `==` para verificar igualdade de valores.  
> Nunca use `is` para comparar números ou strings diretamente.

---

## Operadores de associação

Verificam se um elemento **pertence** a uma sequência.

| Operador | Significado |
|----------|------------|
| `in` | Pertence à sequência |
| `not in` | Não pertence à sequência |

```python
numeros = [1, 2, 3, 4, 5]

print(3 in numeros)      # True
print(6 not in numeros)  # True

curso = "Python"
print("P" in curso)      # True
print("x" not in curso)  # True
```

## Resumo rápido

- Conversão de tipos é o processo de transformar um valor de um tipo em outro.
- Isso é útil quando precisamos manipular dados de forma diferente da original.

## Quando usar

- Quando um dado vem como `string`, mas precisa ser tratado como número
- Quando precisamos exibir um valor num formato específico
- Quando queremos evitar erros em operações

## Observações importantes

- `int()` remove a parte decimal, sem arredondar matematicamente
- `float()` transforma números em decimais
- Nem toda string pode ser convertida para número

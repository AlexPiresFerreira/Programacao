
## Resumo rápido

- Em Python, os tipos de dados definem como um valor é armazenado e quais operações podem ser feitas com ele.
- Python possui vários tipos built-in, como texto, números, listas, dicionários e booleanos.

## Tipos built-in

- **Texto:** `str`
- **Numérico:** `int`, `float`, `complex`
- **Sequência:** `list`, `tuple`, `range`
- **Mapa:** `dict`
- **Coleção:** `set`, `frozenset`
- **Booleano:** `bool`
- **Binário:** `bytes`, `bytearray`, `memoryview`

## Observações importantes

- `bool` é uma subclasse de `int`
- Em Python, `0` pode representar `False`
- Qualquer número diferente de `0` pode ser interpretado como `True`

### Numéricos

#### Inteiro (`int`)
- Representa números sem casas decimais.
- Pode ser positivo, negativo ou zero.
- Suporta separador visual `_` para melhorar a leitura.

```python
numero = 8
negativo = -8
positivo = +8

# separador visual (não afeta o valor)
grande = 11_111_111   # equivale a 11111111
```

#### Ponto flutuante (`float`)
- Representa números com casas decimais.
- Suporta notação científica com `E`.

```python
preco = 8.5
velocidade_luz = 3E8   # equivale a 3 x 10^8 = 300000000.0
```

#### Octais
- Definidos com o prefixo `0o`.
- Representam números na base 8.

```python
octal = 0o123
```

#### Hexadecimais
- Definidos com o prefixo `0x`.
- Representam números na base 16.

```python
hexa = 0x123
```

---

### String (`str`)

- Representa texto.
- Pode ser definida com aspas simples ou duplas.

```python
nome = "Python"
linguagem = 'Python'
```

#### Caracteres de escape

| Sequência | Significado |
|-----------|-------------|
| `\n` | Pula linha |
| `\\` | Barra invertida literal `\` |
| `\"` | Aspas duplas dentro de string |
| `\'` | Aspas simples dentro de string |

```python
print("Linha 1\nLinha 2")
print("Caminho: C:\\Users\\Alex")
print("Ele disse: \"Olá!\"")
```

---

### Booleano (`bool`)

- Representa verdadeiro ou falso.
- Apenas dois valores possíveis.

```python
ativo = True
desligado = False
```

## Continuação de linha com `\`

- O caractere `\` ao final de uma linha indica que o código continua na próxima linha.
- Útil para evitar linhas muito longas.

```python
total = 1 + 2 + 3 + \
        4 + 5 + 6
```

## Verificação de tipos com `type()`

- A função `type()` retorna o tipo do objeto passado como argumento.
- Combinada com `is`, permite verificar se uma variável é de um tipo específico.

### Sintaxe

```python
type(variavel) is int
type(variavel) is str
type(variavel) is float
type(variavel) is bool
type(variavel) is list
```

### Exemplos

```python
# verificando tipos diretamente
print(type(1))          # <class 'int'>
print(type(1.5))        # <class 'float'>
print(type("Python"))   # <class 'str'>
print(type(True))       # <class 'bool'>
print(type([1, 2, 3]))  # <class 'list'>
```

```python
# verificando se a variável é de um tipo específico
x = 10

print(type(x) is int)    # True
print(type(x) is float)  # False
print(type(x) is str)    # False
```

```python
# uso prático com condicional
valor = "123"

if type(valor) is str:
    print("É uma string, precisa converter.")
    valor = int(valor)

print(valor + 1)  # 124
```

### `type()` vs `isinstance()`

- `type()` → verifica o tipo **exato**.
- `isinstance()` → verifica se o objeto é de um tipo **ou de uma subclasse** dele.

```python
x = True

print(type(x) is bool)       # True
print(type(x) is int)        # False

print(isinstance(x, bool))   # True
print(isinstance(x, int))    # True  → bool é subclasse de int
```

> Para a maioria dos casos práticos, prefira `isinstance()`.

# Python - Tipos e Variáveis

## Resumo rápido
- Python trabalha com vários tipos de dados prontos.
- Variáveis armazenam valores e podem mudar ao longo do programa.
- Constantes seguem convenção de nomeação, mesmo que Python não tenha suporte nativo a constantes.
- A função `input()` lê valores da entrada padrão como `str`.

---

## Tipos de dados

Python possui diversos tipos built-in:

| Categoria  | Tipo(s) | Exemplo |
|-----------|---------|---------|
| Texto     | `str` | `"Texto"`, `'Texto'`, `'''Texto'''`, `"""Texto"""` |
| Numérico  | `int`, `float`, `complex` | `10`, `10.5` |
| Sequência | `list`, `tuple`, `range` | `[1, 2]`, `(1, 2)`, `range(10)` |
| Mapa      | `dict` | `{"nome": "Alex"}` |
| Coleção   | `set`, `frozenset` | `{1, 2, 3}` |
| Booleano  | `bool` | `True`, `False` |
| Binário   | `bytes`, `bytearray`, `memoryview` | dados binários |

---

## Convertendo tipos

- Em muitos casos, precisamos converter um valor para outro tipo.
- Isso é feito com construtores como `int()`, `float()`, `str()`, etc.

### Exemplo: `float` para `int`

```python
preco = 10.30
print(preco)
# 10.3

preco = int(preco)
print(preco)
# 10
```

### Exemplo: `int` para `float`

```python
preco = 10
print(float(preco))
# 10.0
```

### Exemplo: `int` para `str`

```python
idade = 28
print(str(idade))
# "28"
```

### Exemplo: `str` para número

```python
preco = "10.50"
idade = "28"

print(float(preco))
# 10.5

print(int(idade))
# 28
```

### Erro de conversão
- Se a string não representar um número válido, ocorre `ValueError`.

```python
preco = "python"
print(float(preco))
# ValueError
```

---

## Variáveis

- Variável é um nome que aponta para um valor.
- A forma geral é:

```python
nome_variavel = valor
```

### Boas práticas de nomeação
- Use **snake_case**.
- Escolha nomes **sugestivos**.

```python
limite_saque_diario = 1000
nome_completo = "Alex Silva"
```

### Exemplo

```python
age = 23
name = "Guilherme"

print(f"Meu nome é {name} e eu tenho {age} ano(s) de idade.")
```

### Reatribuição

- A variável pode receber um novo valor.

```python
age = 28
name = "Guilherme"

print(f"Meu nome é {name} e eu tenho {age} ano(s) de idade.")

age = 27
name = "Giovanna"

print(f"Meu nome é {name} e eu tenho {age} ano(s) de idade.")
```

---

## Constantes

- Constante é um valor que não deveria mudar durante a execução do programa.
- Python **não tem constantes nativas**.
- Por convenção, usamos nomes em **maiúsculo**.

```python
BRAZILIAN_STATES = ["SP", "RJ", "MG"]
ABS_PATH = "/home/alex/documentos/"
DEBUG = True
AMOUNT = 30.2
```

### Boas práticas
- Nome de constante em **caixa alta**.
- Nome de variável em **snake_case**.

---

## Função `input()`

- `input()` lê dados da entrada padrão.
- O valor retornado por `input()` é sempre uma **string**.

### Sintaxe

```python
variavel = input("Mensagem: ")
```

### Exemplo

```python
nome = input("Informe o seu nome: ")
print(nome)
```

### Observação importante
- Mesmo quando o usuário digita números, o retorno continua sendo `str`.

```python
idade = input("Informe sua idade: ")
print(type(idade))  # <class 'str'>
```

- Se precisar fazer contas, converta com `int()` ou `float()`.

---

## Função `type()`

- `type()` mostra o tipo de uma variável ou valor.

```python
print(type(10))      # <class 'int'>
print(type(10.5))    # <class 'float'>
print(type("abc"))   # <class 'str'>
```

---

## Resumo final

- Python possui vários tipos built-in: `str`, `int`, `float`, `list`, `dict`, `set`, `bool`, etc.
- Conversões são feitas com `int()`, `float()`, `str()`, entre outras.
- Variáveis seguem o padrão `snake_case`.
- Constantes seguem o padrão em maiúsculo.
- `input()` sempre retorna uma string.

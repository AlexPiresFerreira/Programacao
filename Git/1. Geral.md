
O **Git** é um **sistema de controle de versão** usado para gerenciar alterações em arquivos, principalmente em projetos de código-fonte.  
Ele permite acompanhar versões, colaborar com outras pessoas e evitar perda de trabalho quando várias mudanças acontecem ao mesmo tempo.

## Para que serve

- Controlar versões de arquivos e código
- Facilitar trabalho em equipe
- Registrar histórico de alterações
- Permitir voltar para versões anteriores
- Organizar desenvolvimento com **branches**



- **Repositório**
    
    - Local onde o projeto e seu histórico ficam armazenados
- **Branch**
    
    - Linha paralela de desenvolvimento
    - Serve para criar versões diferentes do projeto sem afetar a principal
- **Merge**
    
    - Junta alterações de uma branch em outra
- **Commit**
    
    - Salva uma alteração no histórico do Git
- **Push**
    
    - Envia as alterações do repositório local para o remoto
- **Pull**
    
    - Baixa e atualiza alterações do repositório remoto
- **SSH**
    
    - Forma segura de autenticação para trabalhar com repositórios remotos


#### Entrar na pasta do vault

```
cd caminho/para/seu/vault
```

#### Inicializar o repositório Git

```
git init
```

#### Adicionar o repositório remoto do GitHub

```
https://github.com/seu-usuario/meu-vault-obsidian.git
```

#### Criar um .gitignore para ignorar arquivos desnecessários

```
echo ".obsidian/workspace.json" >> .gitignore
echo ".obsidian/workspace-mobile.json" >> .gitignore
echo ".trash/" >> .gitignore
echo ".DS_Store" >> .gitignore
```
#### Primeiro commit

```
git add .
git commit -m "primeiro commit: vault inicial"
git branch -M main
git push -u origin main
```

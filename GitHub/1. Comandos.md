
#### Configuração de name e e-mail

- git config --global user.name "nome"
- git config --global  user.email "e-mail"

#### Autenticação

- Chave ssh
	- ssh-keygen
	- .pub >> publica
	- cat <chave.pub>
	- git remote set-url origin <URL SSH>

#### Ver as configurações

- git config --list

#### Verificação da versão

- git --version

#### Mandar o conteudo

 - git push -u origin main


- Operações lógicas definem como os fluxos do programa se comportam diante de proposições.
- **Proposição** é uma afirmação que pode ser avaliada como verdadeira ou falsa.
- Exemplo:
    - “Brasília é a capital do Brasil.”

### Tipos de operações lógicas

- **Negação**
    
    - Inverte o valor lógico de uma proposição.
    - Símbolo comum: `~`
- **Conjunção**
    
    - Só é verdadeira quando **ambas** as proposições são verdadeiras.
    - Símbolo comum: `^`
- **Disjunção**
    
    - É verdadeira se **pelo menos uma** proposição for verdadeira.
    - Símbolo comum: `v`
- **Disjunção exclusiva**
    
    - Verdadeira apenas quando **uma** das proposições é verdadeira.
- **Condicional**
    
    - Expressa a ideia de “se… então…”.
    - Símbolo comum: `→`
- **Bicondicional**
    
    - Duas proposições dependem uma da outra.
    - Símbolo comum: `<->`

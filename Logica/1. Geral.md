
# Lógica

A **Lógica** estuda a forma como os problemas são organizados e resolvidos por meio de **proposições, operadores e estruturas de controle**.  
Ela é a base para construir algoritmos, tomar decisões e definir o fluxo de execução de um programa.

## 1. Conceitos gerais

- **CamelCase**
    
    - Forma de escrita em que palavras são unidas e cada nova palavra começa com letra maiúscula.
    - Ex.: `oneTwoThreeFour`
- **Versionamento semântico**
    
    - Padrão de versões como `1.0.1`
    - Normalmente representa:
    - **MAJOR**: mudanças grandes
    - **MINOR**: novas funcionalidades
    - **PATCH**: correções pequenas
- **Algoritmo**
    
    - Sequência lógica de passos para resolver um problema.

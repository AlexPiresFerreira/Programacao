
## Extensões

### Python
- **Python**
     - Extensão principal para desenvolvimento em Python no VS Code.
    - Oferece **autocomplete**, **execução de código**, **depuração**, **linting** e suporte a **ambientes virtuais**.
- **autoDocstring - Python Docstring Generator**
     - Gera **docstrings automaticamente** para funções, classes e métodos.
    - Ajuda a documentar o código de forma rápida e padronizada.
- **GitHub Copilot Chat**
     - Assistente com IA para **ajudar a escrever, explicar e revisar código**.
    - Útil para tirar dúvidas, gerar trechos de código e entender erros.
### Markdown
- **Markdown Preview Enhanced**
     - Melhora a **pré-visualização de arquivos Markdown**.
    - Suporta recursos avançados como tabelas, diagramas, fórmulas e exportação mais completa.
- **Markdown All in One**
    - Conjunto de ferramentas para editar Markdown com mais agilidade.
    - Inclui **atalhos**, **sumário automático**, **autoformatação** e outras facilidades.
- **Markdown Paste**
     - Facilita colar conteúdo, especialmente **imagens**, dentro de arquivos Markdown.
    - Pode automatizar o processo de salvar a imagem e criar o link no texto.
- **Markdown PDF**
     - Exporta arquivos `.md` para **PDF**.
    - Útil para compartilhar notas, resumos e documentação em formato final.
